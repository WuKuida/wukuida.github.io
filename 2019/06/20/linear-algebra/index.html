第一讲：行列式的性质

1. 行列式

性质：

  1. 某行（列）加上或减去另一行（列）的几倍，行列式不变
    | 1 2 3 | | 1 2 3 | | 1 2 3 | | 1 2 3 |
    | 2 3 4 | = | 0 -1 -2 | = | 0 -1 -2 | = | 0 -1 -2 | = 1 -1 1 = 1
    | 4 5 7 | | 4 5 7 | | 0 -3 -5 | | 0 0 1 |

  2. 某行（列）乘k，等于k乘此行列式
    | 1 2 3 | | 2 4 6 |
    | 2 3 4 | * 2 = | 2 3 4 |
    | 4 5 7 | | 4 5 7 |

  3. 互换两行（列），行列式变号
    | 1 2 3 | | 4 5 6 |
    | 2 3 4 | * -1 = | 2 3 4 |
    | 4 5 7 | | 1 2 3 |
